This website was designed based on the course requirements, to teach how to display the menu, and how to display meals in a well design web pages, How to work on the UI, How to make the website responsive, How to redirect to different pages, How to create a navigation bar, And much more...

On the main page, instead of your average image carousel, I built a full-page tile board that scales with your screen size and a service map powered by Google Maps, the order being customizable. The food background images fade as you hover.

You can view the live website and see the scrolling animations in action at http://restaurant-ch.com.
